🖥️ Hardware အကြောင်း
Intel vs AMD CPU - ဘယ်ဟာက ပိုကောင်းသလဲ?

CPU ဝယ်ယူတဲ့အခါ ကိုယ်အသုံးပြုမယ့် လုပ်ငန်းစဉ်အပေါ် မူတည်ပြီး ရွေးချယ်သင့်ပါတယ်-

BrandProduct Seriesအားသာချက်
Intel Corei3, i5, i7, i9Single-core performance ကောင်းပြီး Stability မြင့်မားသည်👌။
AMD RyzenR3, R5, R7, R9Multi-core performance ကောင်းပြီး ဈေးနှုန်းနှင့် တန်ဖိုးမျှတသည်😎။

စနစ်တကျ ရွေးချယ်နည်း-

  • AMD 'G' Series: Built-in GPU ပါဝင်သောကြောင့် Graphics Card မပါဘဲ ဂိမ်းဆော့လိုသူများအတွက် အထူးသင့်လျော်သည်။
  • Intel 'F' Series: Graphics Card သီးသန့်သုံးမည့်သူများအတွက် ဈေးသက်သာသော ရွေးချယ်မှုဖြစ်သည်။
  • Socket သတိပြုရန်: Intel အတွက် LGA socket နှင့် AMD အတွက် AM4/AM5 socket မတူညီသောကြောင့် Motherboard ဝယ်လျှင် သေချာစစ်ဆေးပါ။
CPU - Core နှင့် Thread ဆိုတာဘာလဲ?

CPU ကို ကွန်ပျူတာရဲ့ ဦးနှောက်လို့ တင်စားနိုင်ပြီး Core နဲ့ Thread တွေက ဦးနှောက်ရဲ့ လုပ်ဆောင်နိုင်စွမ်းကို ဆုံးဖြတ်ပါတယ်🧠

အမျိုးအစားCores/Threadsအသုံးပြုရန် အသင့်တော်ဆုံး
Entry Level2 Cores / 4 Threadsရုံးသုံး၊ ကျောင်းသုံး၊ အင်တာနက်ကြည့်ရန်🫡။
Mid-Range6 Cores / 12 ThreadsGaming နှင့် Video Editing အသင့်အတင့်လုပ်ရန်😎။
High-End8+ Cores / 16+ ThreadsProfessional Rendering နှင့် Heavy Multitasking။😈

မှတ်ချက်: Core က ရုပ်ပိုင်းဆိုင်ရာ အစိတ်အပိုင်းဖြစ်ပြီး Thread ကတော့ Core ကို အလုပ်လုပ်ဖို့ ခိုင်းစေတဲ့ virtual ညွှန်ကြားချက်တွေ ဖြစ်ပါတယ်။

Storage - HDD နှင့် SSD ဘာကွာသလဲ?

ကွန်ပျူတာရဲ့မှတ်ဉာဏ်ဖြစ်တဲ့ Storage အမျိုးအစားတွေကို နှိုင်းယှဉ်ပြထားပါတယ်-

အမျိုးအစားSpeedအားသာချက်
HDDအနှေးဆုံး🐢ဈေးသက်သာပြီး ဖိုင်အများကြီးသိမ်းနိုင်ခြင်း🫡။
SATA SSDမြန်ဆန်သည်😎HDD ထက် ၁၀ ဆကျော် ပိုမြန်ပြီး တည်ငြိမ်ခြင်း😈။
NVMe SSDအမြန်ဆုံး💀Video Editing နှင့် High-end Gaming အတွက် အကောင်းဆုံး😎👌။
RAM အကြောင်း

RAM (Memory) နည်းပညာက အချိန်နဲ့အမျှ ပိုမြန်လာပြီး ပါဝါစား သက်သာလာပါတယ်ခင်ဗျ

အမျိုးအစားDescriptionStatus
DDR1အစောဆုံး နည်းပညာဖြစ်ပြီး ယခုခေတ်မှာ မသုံးတော့ပါ😭။Legacy ❌
DDR2Speed ပိုမြန်လာသော်လည်း ယနေ့ခေတ်ဂိမ်းများအတွက်က💀။Legacy ❌
DDR3PC အဟောင်းအချို့မှာ မြင်တွေ့နိုင်ဆဲဖြစ်ပြီး Speed နှေးပါသည်‌‌/ပေါ့ပေါ့ပါးပါးဂိမ်းတွေတော့ဆော့လို့ရ😎။Old ⚠️
DDR4လက်ရှိမှာ လူသုံးအများဆုံးဖြစ်ပြီး Speed နှင့် ဈေးနှုန်း မျှတပါသည်😎👌။Standard ✅
DDR5အသစ်ဆုံးနှင့် အမြန်ဆုံးဖြစ်ကာ High-end PC များအတွက် ဖြစ်သည်😈။Premium 🔥

မှတ်ချက်: DDR အမျိုးအစား တစ်ခုနဲ့တစ်ခုဟာ Slot ပုံစံမတူတဲ့အတွက် Motherboard က ထောက်ပံ့တဲ့ အမျိုးအစားကိုပဲ ဝယ်သင့်ပါတယ်🫡။

PSU (Power Supply) - ဝယ်ယူလျှင် ဘာတွေကြည့်ရမလဲ?

PSU က ကွန်ပျူတာအစိတ်အပိုင်း အားလုံးကို လျှပ်စစ်ဓာတ်အား ဖြန့်ဝေပေးတာဖြစ်လို့ အရည်အသွေးကောင်းဖို့ အလွန်အရေးကြီးပါတယ်-

၁။ 80 Plus Efficiency Rating (စွမ်းဆောင်ရည် အဆင့်ဆင့်)

RatingEfficiencyမှတ်ချက်
80+ White80%အခြေခံအဆင့် (Entry Level)
80+ Bronze82% - 85%လူသုံးအများဆုံးနှင့် စိတ်ချရသည်
80+ Gold87% - 90%High-end PC များအတွက် အကောင်းဆုံး

၂။ လိုအပ်သော Wattage (ဝပ်) ပမာဏ

PC TypeSpecs (CPU + GPU)Recommended Wattage
Entry PCi3 / R3 + Integrated GPU450W - 550W
Gaming PCi5 / R5 + Mid-range GPU550W - 650W
Workstationi7 / R7 + High-end GPU750W နှင့်အထက်

အရေးကြီးသော သတိပေးချက်: နာမည်မရှိသော တန်ဖိုးနည်း PSU (Generic PSU) များသည် လျှပ်စစ်အား မတည်ငြိမ်ပါက ကွန်ပျူတာအစိတ်အပိုင်းများကို လောင်ကျွမ်းစေနိုင်သဖြင့် နာမည်ကြီး Brand (ဥပမာ - Corsair, Cooler Master, Seasonic) များကိုသာ ရွေးချယ်သင့်ပါသည်။

GPU (Graphics Card) - ဂိမ်းဆော့ဖို့နဲ့ ဗီဒီယိုတည်းဖြတ်ဖို့ ဘယ်လိုရွေးမလဲ?

GPU က ရုပ်ပုံတွေကို ပြသပေးရုံတင်မကဘဲ Video Editing နဲ့ Rendering ပိုင်းမှာလည်း အဓိက ကူညီပေးပါတယ်-

၁။ GPU အမျိုးအစား နှစ်မျိုး

အမျိုးအစားအမည်အသုံးပြုရန် အသင့်တော်ဆုံး
Integrated (cpu မှာပါသည့် graphics)Intel UHD, Radeon Graphicsရုံးသုံး၊ ကျောင်းသုံးနှင့် ပေါ့ပေါ့ပါးပါး ဂိမ်းများအတွက်။
DedicatedNvidia RTX, AMD Radeon RXProfessional Gaming နှင့် 4K Video Editing အတွက်။

၂။ VRAM (Video Memory) လိုအပ်ချက်

VRAM SizeBest ForExample Task
4GB - 6GBBudget Gaming1080p gaming (Low/Med settings)
8GB - 12GBStandard Gaming1440p gaming နှင့် Video Editing
16GB+Pro Work4K Gaming နှင့် 3D Rendering

Encoding Tips: Nvidia ရဲ့ NVENC Encoder က Video Render ဆွဲတဲ့အခါ CPU ကို ဝန်မပိစေဘဲ ပိုမြန်စေတဲ့အတွက် Adobe After Effects နဲ့ Premiere Pro သုံးတဲ့သူတွေအတွက် ပိုသင့်တော်ပါတယ်။ AMD Radeon ကတော့ ဈေးနှုန်းနဲ့ Gaming Performance မျှတမှုမှာ အားသာပါတယ်။

Windows ၁၀/၁၁ တင်နည်း အဆင့်ဆင့် (Step-by-Step)

Windows ကို စနစ်တကျ တင်နိုင်ဖို့အတွက် အောက်ပါ အဆင့်အတိုင်း လုပ်ဆောင်ရပါမယ်-

အဆင့် (Step)လုပ်ဆောင်ရန် အချက်များမှတ်ချက်
၁။ Bootable USB ပြုလုပ်ခြင်းRufus သို့မဟုတ် Windows Media Creation Tool သုံးပြီး 8GB+ USB ထဲသို့ ISO ဖိုင်ထည့်ပါ။File ပျက်နိုင်၍ Backup လုပ်ထားပါ ⚠️
၂။ BIOS/Boot Menu ဝင်ခြင်းစက်စပွင့်ချင်း F2, F12 သို့မဟုတ် Del key ကို နှိပ်ပြီး USB ကို ပထမဆုံး Boot တက်အောင် ရွေးပါ။Motherboard အလိုက် Key ကွဲပြားသည်
၃။ Partition ပိုင်းခြင်းWindows တင်မည့် Drive ကို ရွေးချယ်ပြီး New Partition ဆောက်ပါ။Drive မှားမဖျက်မိစေရန် သတိပြုပါ ⛔
၄။ Installation စတင်ခြင်း"Install Now" ကိုနှိပ်ပြီး Windows ကူးယူပြီးသည်အထိ စောင့်ပါ။ စက်အလိုအလျောက် Restart ကျပါလိမ့်မည်။USB ကို Restart ကျချိန်တွင် ဖြုတ်နိုင်သည်

အရေးကြီးသောအချက်: Windows တင်ပြီးပါက Drivers များ (ဥပမာ- Graphics Driver, Chipset Driver) ကို မဖြစ်မနေ Update လုပ်ပေးရပါမယ်။

Linux တင်နည်းနှင့် Windows နှင့် Dual Boot လုပ်နည်း

Windows ကို မဖြုတ်ဘဲ Linux Mint or other os, ကို ဘေးချင်းယှဉ်တွဲသုံးနိုင်မယ့် Dual Boot တင်နည်း အဆင့်ဆင့် ဖြစ်ပါတယ်-

အဆင့် (Step)လုပ်ဆောင်ရန် အချက်များမှတ်ချက်
၁။ Disk Space ဖယ်ခြင်းWindows ရဲ့ "Disk Management" ထဲမှာ Linux အတွက် အနည်းဆုံး 50GB ခန့် Shrink လုပ်ပြီး နေရာလွတ်ဖယ်ပါ။Unallocated Space ဖြစ်နေရပါမယ်
၂။ Bootable USB လုပ်ခြင်းLinux Mint ISO ကို Rufus သို့မဟုတ် BalenaEtcher သုံးပြီး USB ထဲ ထည့်ပါ။Linux Mint Cinnamon edition ကို အကြံပြုပါသည်
၃။ Installation TypeInstall လုပ်စဉ် "Install Linux Mint alongside Windows" ကို ရွေးချယ်ပါ။ဒါက အလွယ်ဆုံးနဲ့ အန္တရာယ်အကင်းဆုံးပါ ✅
၄။ Boot Menuတင်ပြီးသွားရင် စက်စဖွင့်ချင်းမှာ Windows သို့မဟုတ် Linux Mint ကြိုက်ရာရွေးသုံးနိုင်တဲ့ Menu ပေါ်လာပါလိမ့်မယ်။GRUB Bootloader ဟု ခေါ်သည်

Linux Mint ရဲ့ အားသာချက်များ-

  • Customization: Windows ထက် ပိုမိုလှပအောင် စိတ်ကြိုက် ပြင်ဆင်နိုင်ခြင်း။
  • Performance: Windows ထက် ပေါ့ပါးပြီး Software တော်တော်များများကို အခမဲ့ သုံးနိုင်ခြင်း။
  • Stability: Virus အန္တရာယ် နည်းပါးပြီး PC အဟောင်းများမှာပင် မြန်ဆန်ခြင်း။

Hardware ShortRAM slot ထဲ ဖုန်မှုန့်/အညစ်အကြေးများ ဝင်ခြင်း။RAM slot ကို Contact Cleaner ဖြင့် သေချာရှင်းလင်းပါ။
OverheatingCPU အပူချိန် အရမ်းတက်နေခြင်း။Thermal Paste အသစ်ပြန်ထည့်ပြီး Fan များ လည်၊ မလည် စစ်ပါ။
Drive FailureSSD သို့မဟုတ် HDD ပျက်စီးနေခြင်း။Drive ကျန်းမာရေးကို CrystalDiskInfo ဖြင့် စစ်ဆေးပါ။

အရေးကြီးသော သင်ခန်းစာ: PC ကို Thermal Paste အသစ်လဲတဲ့အခါ Motherboard ရဲ့ အစိတ်အပိုင်းတွေထဲ (အထူးသဖြင့် RAM slot) Grease တွေ မဝင်သွားအောင် အထူးသတိထားဖို့ လိုအပ်ပါတယ်။ Grease ဝင်သွားရင် BIOS ထဲမှာတောင် PC Freeze ဖြစ်တတ်ပါတယ်။

⚠️ Safety Tips
လျှပ်စစ်အန္တရာယ် ကာကွယ်ရေး သင်ခန်းစာများ

လျှပ်စစ်ပစ္စည်းတွေ သုံးတဲ့အခါ မတော်တဆမှု မဖြစ်အောင် ဒါတွေကို သတိထားရပါမယ်-

အန္တရာယ်ဖြစ်ပွားနိုင်သည့် အကြောင်းရင်းကာကွယ်နည်း
Charger ပေါက်ကွဲခြင်းအရည်အသွေးမပြည့်မီသော Charger တွင် ပါဝါစားများသည့် USB RGB Lights များ ထိုးမိခြင်း။နာမည်ကြီး Brand (ဥပမာ- Samsung, Apple, Anker) စစ်စစ်များကိုသာ သုံးပါ။
Breaker ကျခြင်းလျှပ်စစ်ပစ္စည်းတစ်ခုခုတွင် Short circuit ဖြစ်နေခြင်း။Breaker ခဏခဏ ကျနေပါက မသင်္ကာသော ပစ္စည်းကို ချက်ချင်းဖြုတ်ပါ။
Hardware လောင်ကျွမ်းခြင်းအရည်အသွေးညံ့သော PSU (Power Supply) သုံးစွဲခြင်း။80 Plus Rating ပါဝင်သော PSU ကိုသာ သုံးစွဲပါ။

ကျနော်ကိုယ်တွေ့သင်ခန်းစာ: 45W Charger တစ်ခုကို RGB Light ထိုးမိရာကနေ ပေါက်ကွဲပြီး အိမ်ရဲ့ Main Breaker ပါ ကျသွားတဲ့အထိ အန္တရာယ်ရှိနိုင်ပါတယ်။ ဒါကြောင့် ပါဝါအထွက် (Wattage) နဲ့ ပစ္စည်းရဲ့ အရည်အသွေးကို အမြဲစစ်ဆေးပါ။

ကွန်ပျူတာ အသုံးပြုရာတွင် သတိပြုရန်များ
  • Liquid: ကွန်ပျူတာအနီးတွင် ရေခွက်၊ ကော်ဖီခွက်များ မထားပါနှင့်။
  • Cleaning: PC ကို သန့်ရှင်းရေးလုပ်လျှင် ပါဝါပိတ်ရုံတင်မကဘဲ ပလပ်ပါ ဖြုတ်ထားရပါမည်။
  • Thunder: မိုးကြိုးပစ်နေချိန်တွင် ကွန်ပျူတာနှင့် အင်တာနက် Router များကို ခေတ္တပိတ်ထားသင့်သည်။
